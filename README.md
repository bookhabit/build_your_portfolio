<h1>기술스택</h1>
<h2>Frontend</h2> 
<h3>Core</h3>
<p>React , typescript</p>

<h3>스타일링</h3>
<p>tailwindcss</p>
<p>애니메이션 - css, gsap</p>

<h1>Project Introduce</h1>
<p>포트폴리오를 만드는 개발자 또는 취준생들을 위해서 이력서와 포트폴리오 form을 제공하고 작성한 form에 맞춰서 포트폴리오를 꾸며주는 웹사이트입니다.
사용자는 제공된 form을 등록하고 주어진 디자인을 선택하고 변경하면서 '사용'만 하면 됩니다. </p>
